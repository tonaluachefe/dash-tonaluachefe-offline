# Move NFT Contract

Este módulo Move implementa um contrato simples de NFT para o Sui Bootcamp.

## Estrutura

- `Move.toml` - Configuração do pacote Move
- `sources/my_nft.move` - Contrato principal com struct MyNFT e função mint

## Como compilar

```bash
sui move build
```

## Como testar

```bash
sui move test
```

## Como fazer deploy

Veja o arquivo `DEPLOY.md` na raiz do projeto para instruções completas.
